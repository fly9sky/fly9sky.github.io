# 自序

> 太史公以余万字而述数千年历史，余身嘉之。后历代均有史书，然其卷佚浩繁甚为不便，二十四十计四千万言，于效太史公择其要成此《续史记》。

> 太史公已列传纪者则不书。

# 本纪

> > 基本上是编年体，兼述帝王本人事迹。有帝王之实而无帝王之名者，如项羽、吕雉，亦可列入本纪（见司马迁《史记》）。

> >  大一统王朝方有纪，太史公以著秦本纪，故于自汉起叙之。
